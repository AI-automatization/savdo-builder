## Кого таргетим в Phase A (ICP — Ideal Customer Profile)

Из 4 B2B-персон плана для Phase A берём тех, кого **легче всего убедить и онбордить**:

1. **Telegram-channel seller** (канал 1–50k, уже продаёт в TG) — самый горячий:
   у них уже есть аудитория и боль «заказы в личке теряются». **Приоритет #1.**
2. **Instagram-микро-продавец** — хочет упорядочить flow, но аудиторию надо тащить
   в TG. Приоритет #2.
3. **Региональный продавец** (Самарканд/Бухара/Фергана) — Uzum-логистика слабая,
   локальный storefront = ценность. Приоритет #2.
4. **Self-employed/ремесленник** (керамика, сюзани, food-artisan) — аутентика не
   вписывается в маркетплейсы. Тёплые, но низкий объём заказов. Приоритет #3.

**Критерий «берём в beta»:** активно продаёт **сейчас** (есть свежие посты с товаром
и ценами), есть хотя бы ~10 заказов/мес, готов 15 минут на онбординг-звонок.

## Скрипт первого касания (шаблон, RU)

> Адаптировать под канал. Не спам — персонализировать первой строкой (упомянуть
> конкретный товар/пост продавца).

```
Привет, [имя]! Видел ваш канал — [конкретный товар/пост]. 🔥

Делаем maxsavdo — это магазин прямо в Telegram: каталог, корзина, заказы и
статусы вместо переписки в личке. Без комиссии как у Uzum — просто подписка,
а сейчас, на запуске, — бесплатно.

Можно за 5 минут собрать ваш storefront и дать ссылку, которую вы кидаете
подписчикам. Покажу как это выглядит на вашем товаре?
```

### UZ-вариант — первое касание (черновик, Азим сверяет нативно)

> Орфография: oʻ/gʻ через ʻ (U+02BB), тутук-белгиси (`taʼminlash`, `maʼlumot`) через ʼ (U+02BC).
> Первую строку **персонализировать** — упомянуть конкретный товар/пост продавца. Не спам.

```
Assalomu alaykum, [ism]! Kanalingizni koʻrdim — [aniq mahsulot/post]. 🔥

Biz maxsavdoni ishlab chiqyapmiz — bu toʻgʻridan-toʻgʻri Telegram ichidagi
doʻkon: katalog, savat, buyurtmalar va ularning holati — lichkadagi
yozishmalar oʻrniga. Uzumdagidek komissiyasiz — oddiy obuna, hozir esa,
ishga tushirish chogʻida — mutlaqo bepul.

Doʻkoningizni 5 daqiqada yigʻib beramiz va obunachilaringizga tashlaydigan
havola olasiz. Aynan sizning mahsulotingizda qanday koʻrinishini koʻrsataymi?
```

### UZ — follow-up (если не ответили через 2–3 дня)

```
Assalomu alaykum, [ism]! Oldingi xabarimni eslatib oʻtaman 🙂
maxsavdo'da doʻkon ochish hozir bepul, sozlash 5 daqiqa. Xohlasangiz,
tayyor namunani sizning bir mahsulotingizda koʻrsatib beraman — keyin
oʻzingiz qaror qilasiz. Qiziqmi?
```

### UZ — для TG-чатов продавцов (мягко, value-first, без прямого оффера)

```
Lichkada buyurtma qabul qilib charchaganlar bor-a? 😅 Biz Telegram ichida
toʻliq doʻkon qilyapmiz — katalog, savat, buyurtma statuslari. Uzum
komissiyasisiz. Beta hozir bepul — kimga qiziq boʻlsa, yozing, koʻrsataman.
```

> **Глоссарий ценностного оффера (UZ):** комиссия → *komissiya*; подписка → *obuna*;
> витрина/storefront → *doʻkon* (или транслит *vitrina*); заказ → *buyurtma*; доставка →
> *yetkazib berish*; самовывоз → *olib ketish*; статус заказа → *buyurtma holati*.
> Сверить с терминологией в `apps/web-seller/src/lib/i18n/uz.ts` (ревью —
> `analiz/audits/uz-translation-review-2026-05-30.md`).

## Воронка Phase A (целевые числа)

```
Контактов отправлено      ─── 80–120
   ↓ ответ ~30%
Ответили                  ─── 25–35
   ↓ конверсия в звонок ~60%
Онбординг-звонок          ─── 15–25
   ↓ активация ~80%
Создали магазин + товар   ─── 20–30   ← ЦЕЛЬ Phase A
```

При conversion хуже — увеличиваем верх воронки (больше контактов), не снижаем
критерий ICP.

## Онбординг-чеклист (15 минут на продавца)

- [ ] Создать магазин (название, slug, лого, cover, описание).
- [ ] Завести 3–5 товаров с фото + ценами + вариантами (показать на их реальном товаре).
- [ ] Настроить доставку (fixed/manual/none) + регионы + самовывоз.
- [ ] Дать storefront-ссылку → продавец постит её в свой канал.
- [ ] Показать seller-dashboard: где приходят заказы, чат, статусы.
- [ ] Включить уведомления в Telegram-бот (@savdo_builderBOT).
- [ ] Договориться о фоллоу-апе через 7 дней (первые заказы? что мешает?).
