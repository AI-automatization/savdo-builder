## Детали по кандидатам

### 1. OlSot (8.95) — был рекомендован, Азим отклонил
- **Этимология:** `ol` (купи) + `sot` (продай) — 2 узбекских глагола слиплись в одно слово.
- **Идея:** 4 буквы, encode marketplace в имени, не из семейства `*_bozor`.
- `olsot.com` — ✅ свободен (WHOIS подтверждён).
- `olsot.uz` — не проверен надёжно (cctld.uz форма на JS, scrape отказал).

### 2. Bizniki (8.10)
- **Этимология:** «наш» (узб) — communal позиционирование.
- `bizniki.com` — ✅ свободен.
- `bizniki.uz` — не проверен.

### 3. Anorbozor (6.95)
- **Этимология:** «гранатовый базар» — национальный фрукт UZ.
- `anorbozor.com` — ✅ свободен.
- `anorbozor.uz` — не проверен (ECONNREFUSED при прямом visit — слабый сигнал что свободен).

### 4. Soddabozor (6.30)
- **Этимология:** «простой базар» (sodda = простой).
- `soddabozor.com` — ✅ свободен.

### 5. Dukonim (6.30 — ослаблен с 6.90)
- **Этимология:** «мой магазин» (узб).
- `dukonim.com` — ✅ свободен.
- `dukonim.uz` — ❌ **взят с 2023** (who.is подтверждено) → название мёртвое.

### 6. Bizbozor (6.15)
- **Этимология:** «наш базар».
- `bizbozor.com` — ✅ свободен.

### 7. Sotbozor (6.10)

- `sotbozor.com` — ✅ свободен.

### 8. Karvonbozor (6.00 — ослаблен с 6.65)

## Вывод Азима (2026-05-21)

> «Эти название не впечатляют, но сохрани»

Замечания (по контексту, не от Азима явно — но из его прошлых ревью):
- `*_bozor` семейство — переполнено, не выделяемся.
- Двусложные/глагольные слипы (OlSot, Sotma) — слишком утилитарны, не emotional.
- Имена-фрукты (Anor, Olcha) — UZ-маркет уже использует.

## Что для Round 2 не повторять
1. Не предлагать `*_bozor` композиты — целая категория не выделяется.
2. Не предлагать имена UZ-fruits / common Uzbek nouns — главные уже у Uzum / Olcha / Asaxiy.
3. Не предлагать словарные узбекские слова без своей trademark-логики — все короткие словарные `.com` разобраны 1999-2020.
4. Не предлагать слипы глаголов (OlSot pattern) — Азим не впечатлён.

## Что для Round 2 попробовать
- **Coined/invented** имена с UZ-фонетикой, но не словарные (низкий риск collision)
- **Англо-узбекские гибриды** с эмоциональной нагрузкой (не утилитарной)
- **Имена с UZ-культурным контекстом** не из еды/природы (например: ремёсла, узоры, история)
- **2-сложные с открытыми гласными** (бренд-friendly: Olo, Aya, Iya, Mira-style)
- Возможно — **обращение к идентичности «marketplace для маленьких сёллеров»** через метафору (не «базар», а другое)
