# Savdo — Brand Guide

> Savdo — e-commerce платформа в Telegram для продавцов Узбекистана.
> «Savdo» (узб.) — торговля, коммерция.

---

## 1. Mission

Дать любому жителю Узбекистана возможность открыть свой магазин в Telegram
за 3 минуты — без сайта, домена, бухгалтерии или фотостудии.

## 2. Positioning

| Аспект | Значение |
|--------|----------|
| **Категория** | UZ-маркетплейс / Telegram-commerce |
| **Главный конкурент** | Uzum Market, OLX UZ, Olcha |
| **Главное отличие** | Полный flow в Telegram (без отдельного приложения / сайта). 0 затрат на старте. |
| **Год запуска** | 2026 |
| **Юрисдикция** | Республика Узбекистан |

## 3. Audience — 3 персоны

### 🛍 Buyer (Покупатель)
- **Возраст:** 18-45, городские
- **Канал входа:** Telegram-канал продавца, Instagram-реклама → deep-link
- **Что важно:** скорость покупки, безопасность, доставка
- **Боли:** UX Uzum громоздкий; OLX без модерации; нет привязки к Telegram

### 🏪 Seller (Продавец)
- **Возраст:** 25-50, малый бизнес
- **Канал входа:** «Сосед открыл магазин, заработал» → @savdo_builderBOT
- **Что важно:** ноль setup'а, простой интерфейс, заказы прямо в TG
- **Боли:** Uzum — высокая комиссия + сложная регистрация; Instagram-магазин — нет автоматизации

### 👨‍💼 Admin (Команда Savdo)
- **Возраст:** internal team (1-3 человека)
- **Канал входа:** OTP через @savdo_builderBOT
- **Что важно:** dense UI, audit trail, MFA, быстрая модерация
- **Боли:** мониторинг 1000+ магазинов с одной dashboard

## 4. Tone of Voice

### Buyer-facing (TMA buyer + web-buyer)
- **Дружелюбный, тёплый, узнаваемо UZ-local**
- Использовать местные обороты: «открыть магазин», «написать продавцу», «доставит сегодня»
- Эмодзи — да (🛍, 🏪, 📦, ✨), но 1-2 на блок
- Никаких корпоратизмов: НЕ «осуществить заказ», ДА «оформить заказ»
- Обращение всегда на «вы»

### Seller-facing (TMA seller + web-seller)
- **Деловой, продуктивный, доверительный**
- Цифры на первом плане: «3 заказа сегодня», «+12% к месяцу»
- Эмодзи минимум — только статусы (✅, ⏳, ❌)
- Обращение на «вы», но более прямо: «Подтвердите заказ», «Привяжите канал»

### Admin-facing
- **Технический, dense, без украшений**
- Иногда английский — где так привычнее: «PENDING», «APPROVED», audit log codes
- Эмодзи только семантические (⚠️ для warning, 🔒 для locked)

### Marketing / external (соцсети, реклама)
- **Энергичный, конкретный, бенефит-first**
- Заголовок = главный бенефит + цифра: «Открой магазин в Telegram за 3 минуты»
- CTA глагольный: «Начать продавать», «Открыть магазин»
- 🇺🇿 / 🛍 / 🏪 — допустимо в hero
- На двух языках: русский + узбекский (Latin)

## 6. Brand Sub-systems (намеренное разделение)

Savdo использует **3 design-системы** для разных аудиторий — это
осознанное решение, не баг. Один parent-brand, три визуальных контекста:

| Контекст | Audience | Design system | Accent |
|----------|----------|---------------|--------|
| **TMA Buyer** | Покупатель в Telegram | Liquid Glass (dark) | Orchid Violet `#A855F7` |
| **TMA Seller** | Продавец в Telegram | Liquid Glass (dark, role-scoped) | Cyan `#22D3EE` |
| **Web Buyer** | Покупатель в браузере (SEO landing) | Soft Color Lifestyle (warm light) | Terracotta `#7C3F2E` |
| **Web Seller** | Продавец в браузере (CRM) | Liquid Authority (cool light/dark) | Violet `#7C3AED` |
| **Admin** | Внутренняя команда | Liquid Authority (cool dark default) | Indigo `#818CF8` |

**Почему 3 системы:**
- TMA — нативный Telegram, dark-by-default, glassmorphism (Apple 2026 Liquid Glass).
- Web-buyer — landing/SEO/marketing страницы, warm-tone привлекает retail-аудиторию.
- Web-seller + Admin — рабочий инструмент, cool-tone снижает усталость.

См. [COLORS.md](COLORS.md) для полных палитр.

## 7. Don'ts

- ❌ НЕ переводить «Savdo» — это и есть бренд (как «Uber» или «Yandex»)
- ❌ НЕ использовать gradient на тексте (читаемость в TG WebView страдает)
- ❌ НЕ ставить punctuation внутри логотипа («Savdo.uz» — только в URL)
- ❌ НЕ использовать обычный апостроф `'` в узбекских текстах — только обратный `ʻ` (U+02BB)
- ❌ НЕ упоминать «Uzum» / «OLX» в Buyer-facing material (это вызывает comparison-mindset)
