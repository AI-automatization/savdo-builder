# Savdo — Design: Мобильное приложение продавца (mobile-seller)

> Контекст для AI-генерации UI/UX мобильного приложения продавца.
> Стиль: Liquid Glass / Glassmorphism. React Native + Expo.

---

## КОНТЕКСТ

**Приложение:** `apps/mobile-seller`
**Стек:** React Native + Expo + NativeWind (Tailwind для RN)
**Платформа:** iOS + Android
**Статус:** Phase 3 (запуск после web версий)
**Язык:** RU + UZ

**Главный принцип:** Продавец управляет заказами с телефона. Подтвердить заказ — не вставая с дивана.
Быстрые уведомления → быстрые действия.

## GLASSMORPHISM ДЛЯ REACT NATIVE

```javascript
// NativeWind + StyleSheet
// Фон экрана
const screenStyle = {
  background: 'linear-gradient(135deg, #0f0c29, #302b63, #24243e)',
  // В RN используем LinearGradient из expo-linear-gradient
};

// Glass-карточки в RN
const glassCard = {
  backgroundColor: 'rgba(255,255,255,0.08)',
  borderWidth: 1,
  borderColor: 'rgba(255,255,255,0.18)',
  borderRadius: 20,
  // BlurView из expo-blur для backdrop-filter
  overflow: 'hidden',
};

// Tailwind классы через NativeWind
className="bg-white/8 border border-white/20 rounded-2xl overflow-hidden"
```

**Компоненты:**
```javascript
// Используем expo-blur для BlurView
import { BlurView } from 'expo-blur';

<BlurView intensity={80} tint="dark">
  <View style={glassCard}>
    {children}
  </View>
</BlurView>

// Градиент фона
import { LinearGradient } from 'expo-linear-gradient';

<LinearGradient colors={['#0f0c29', '#302b63', '#24243e']} style={styles.bg}>
  {children}
</LinearGradient>
```

## UX ПАТТЕРНЫ ДЛЯ МОБАЙЛ

```
✓ Swipe вправо на заказе → "Подтвердить"  (зелёный action)
✓ Swipe влево на заказе  → "Отменить"    (красный action)
✓ Pull-to-refresh на всех списках
✓ Skeleton загрузка (не спиннер)
✓ Haptic feedback при подтверждении заказа
✓ Bottom sheet для деталей (вместо navigate)
✓ Safe area: iOS notch + home indicator + Android nav bar
✓ Tab bar: badge на иконке при новых заказах/сообщениях
✓ Offline: показываем кэшированные данные + баннер "Нет интернета"
```
