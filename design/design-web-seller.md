# Savdo — Design: Дашборд продавца (web-seller)

> Контекст для AI-генерации UI/UX панели управления продавца.
> Стиль: Liquid Glass / Glassmorphism. Desktop-first.

---

## КОНТЕКСТ

**Приложение:** `apps/web-seller`
**Кто видит:** Продавец — управляет своим магазином, товарами, заказами.
**Устройство:** Десктоп / планшет (приоритет), мобайл поддерживается.
**Стек:** Next.js 14 (App Router), Tailwind CSS, DaisyUI v5.
**Язык:** RU + UZ.

**Главный принцип:** Операционная скорость. Подтвердить заказ — 1 клик.
Статусы и счётчики всегда на видном месте.
