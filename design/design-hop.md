# Savdo — Design: Лендинг / Hop Page (savdo.uz)

> Контекст для AI-генерации UI/UX лендинга.
> Стиль: Liquid Glass / Glassmorphism. Фон: тёмный градиент.

---

## ЧТО ТАКОЕ HOP PAGE

Это главная маркетинговая страница savdo.uz — точка входа для продавцов, которые хотят создать свой онлайн-магазин. Покупатели сюда не попадают (у них есть `/store/[slug]`).

**Цель страницы:** Продавец заходит → понимает что это → регистрируется.
**Аудитория:** Мелкий продавец из Узбекистана, продаёт через Telegram/Instagram, хочет выйти на новый уровень.
**Язык:** RU + UZ (переключатель в шапке).

---

## СТРУКТУРА СТРАНИЦЫ (сверху вниз)

### 1. Navbar (sticky)
```
[Логотип Savdo]          [RU | UZ]   [Войти]   [Создать магазин →]
```
- Стекло: `bg-white/5 backdrop-blur-xl border-b border-white/10`
- CTA кнопка: градиентная (синий → фиолетовый)

---

### 2. Hero Section
```
Центрированный текст на весь экран.

Заголовок (H1, 56-72px):
  "Твой магазин в Telegram — уже сегодня"
  или "Продавай умнее. Без лишних хлопот."

Подзаголовок (18px, opacity 0.65):
  "Создай онлайн-магазин за 5 минут.
   Поделись ссылкой в Telegram — покупатели уже в очереди."

CTA кнопки:
  [Создать магазин бесплатно →]   [Посмотреть демо]

Под кнопками: "Уже 500+ продавцов в Ташкенте"
              (аватарки 5 продавцов + число)

Визуал:
  Мокап телефона справа — открыт магазин в Telegram браузере.
  Карточки товаров "плавают" вокруг телефона (glass-эффект).
  Фоновые blobs: синий (#63b3ed) и фиолетовый (#a78bfa), blur 120px.
```

### 4. Как это работает (3 шага)
```
Заголовок: "Запустить магазин — проще, чем кажется"

Шаг 1:  [Иконка 🏪]
         "Создай магазин"
         Заполни название, загрузи логотип,
         добавь товары с фото и ценами.

Шаг 2:  [Иконка 🔗]
         "Поделись ссылкой"
         Отправь savdo.uz/твой-магазин
         в Telegram, Instagram, WhatsApp.

Шаг 3:  [Иконка 💰]
         "Получай заказы"
         Покупатели оформляют заказ онлайн.
         Ты подтверждаешь — они платят при получении.

Каждый шаг — glass-карточка с номером (01 / 02 / 03) крупным шрифтом.
Между шагами — стрелки или пунктирная линия.
```

## ВИЗУАЛЬНЫЙ СТИЛЬ

```css
/* Фон всей страницы */
background: linear-gradient(135deg,
  #0f0c29 0%,
  #302b63 40%,
  #24243e 70%,
  #0f0c29 100%
);

/* Hero blobs (position: absolute, z-index: 0) */
.blob-blue   { background: #63b3ed; filter: blur(120px); opacity: 0.25; }
.blob-purple { background: #a78bfa; filter: blur(120px); opacity: 0.20; }

/* Секции чередуются: тёмный фон / чуть светлее */
.section-alt { background: rgba(255,255,255,0.02); }
```

**Акцент-цвет CTA:** `linear-gradient(135deg, #63b3ed, #a78bfa)`
**Текст основной:** `rgba(255,255,255,0.90)`
**Текст вторичный:** `rgba(255,255,255,0.55)`
**Заголовки:** `font-weight: 700`, `letter-spacing: -0.03em`

## АНИМАЦИИ

```
Hero: fade-in + translateY(-20px → 0) при загрузке
Карточки возможностей: staggered появление при scroll (intersection observer)
CTA кнопка: pulse-glow эффект каждые 3 сек
Navbar: blur усиливается при скролле
Мокап телефона: лёгкое floating (translateY ±8px, 4s ease-in-out)
```
