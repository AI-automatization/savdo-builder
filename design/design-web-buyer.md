# Savdo — Design: Витрина покупателя (web-buyer)

> Контекст для AI-генерации UI/UX покупательской витрины.
> Стиль: Liquid Glass / Glassmorphism. Mobile-first.

---

## КОНТЕКСТ

**Приложение:** `apps/web-buyer`
**Кто видит:** Покупатель — переходит по Telegram-ссылке продавца.
**Устройство:** 80% — мобильный телефон, часто медленный 3G.
**Стек:** Next.js 14 (App Router), Tailwind CSS, DaisyUI v5.
**Язык:** RU + UZ.

**Главный принцип:** Быстро → Просто → Без лишней регистрации.
Цель: от перехода по ссылке до оформленного заказа — максимум 3 клика.

## СТРАНИЦЫ И ЭКРАНЫ

### `/store/[slug]` — Витрина магазина

**Что показывает:**
```
┌─────────────────────────────┐
│  [Обложка магазина — фото]  │
│                              │
│  [Лого]  Название магазина  │
│          ★ Telegram | 📞    │
│          Описание магазина  │
│                              │
│  [Все] [Одежда] [Аксессуары]│  ← фильтр категорий (горизонтальный скролл)
│                              │
│  ┌──────┐  ┌──────┐         │
│  │ фото │  │ фото │         │  ← сетка товаров 2 колонки
│  │      │  │      │
│  │Название  │Название       │
│  │45 000 сум│50 000 сум     │
│  │В наличии │Нет в наличии  │
│  └──────┘  └──────┘         │
│                              │
│  [Написать в Telegram 💬]   │  ← sticky внизу
└─────────────────────────────┘
```

**Компоненты:**
- `StoreCover` — обложка + оверлей с инфо магазина (glass-карточка поверх фото)
- `CategoryFilter` — горизонтальный скролл чипов, активный = градиентный фон
- `ProductGrid` — сетка 2 колонки на мобайле, 3-4 на десктопе
- `ProductCard` — фото (aspect-ratio 1:1), название, цена, бейдж наличия
- `TelegramFAB` — floating кнопка "Написать продавцу" (sticky bottom)

**Glass-стиль для карточки товара:**
```css
.product-card {
  background: rgba(255,255,255,0.08);
  backdrop-filter: blur(16px);
  border: 1px solid rgba(255,255,255,0.15);
  border-radius: 20px;
  overflow: hidden;
}
/* Цена — выделена градиентным текстом */
.price { background: linear-gradient(135deg, #63b3ed, #a78bfa); -webkit-background-clip: text; }
/* Нет в наличии — серый оверлей поверх фото */
.out-of-stock-overlay { background: rgba(0,0,0,0.50); }
```

**Что показывает:**

```
│                              │
│  Название товара             │
│  45 000 сум  ~~50 000 сум~~ │  ← цена + зачёркнутая старая
│                              │
│  Размер:                     │
│  [S]  [M]  [L●] [XL]        │  ← выбор варианта (чипы)
│                              │
│  Цвет:                       │
│  [●Чёрный●] [○Белый] [○Красный]
│                              │
```

**Компоненты:**
- `PhotoGallery` — swipe галерея, zoom по tap, точки внизу
- `VariantSelector` — группы чипов (Размер / Цвет), выбранный = градиентная обводка
- `ColorDot` — цветной кружок с галочкой при выборе
- `PriceDisplay` — текущая цена крупно + зачёркнутая серым
- `AddToCartBar` — sticky bottom bar: кол-во (−/+) + большая кнопка "В корзину"

```html

               text-white font-medium transition-all">

<!-- VariantChip неактивный -->
```
