# Savdo — Design: Мобильное приложение покупателя (mobile-buyer)

> Контекст для AI-генерации UI/UX мобильного приложения покупателя.
> Стиль: Liquid Glass / Glassmorphism. React Native + Expo.

---

## КОНТЕКСТ

**Приложение:** `apps/mobile-buyer`
**Стек:** React Native + Expo + NativeWind
**Платформа:** iOS + Android
**Статус:** Phase 3 (запуск после web версий)
**Язык:** RU + UZ

**Главный принцип:** Ещё быстрее чем web-витрина. Нативные анимации. Offline-первый подход.
Покупатель открывает приложение → сразу видит свои заказы и любимые магазины.

**Ключевое отличие от web-buyer:** В приложении покупатель авторизован, видит историю заказов, избранное, уведомления.

## GLASSMORPHISM ДЛЯ REACT NATIVE

```javascript
// expo-linear-gradient + expo-blur
import { LinearGradient } from 'expo-linear-gradient';
import { BlurView } from 'expo-blur';

// Фон экрана
<LinearGradient
  colors={['#1a1a2e', '#16213e', '#0f3460']}
  style={StyleSheet.absoluteFillObject}
/>

// Glass карточка товара
<BlurView intensity={60} tint="dark" style={styles.productCard}>
  <View>{/* контент */}</View>
</BlurView>

const styles = StyleSheet.create({
  productCard: {
    borderRadius: 20,
    borderWidth: 1,
    borderColor: 'rgba(255,255,255,0.18)',
    overflow: 'hidden',
    backgroundColor: 'rgba(255,255,255,0.08)',
  },
  // Tab Bar
  tabBar: {
    backgroundColor: 'rgba(15,12,41,0.92)',
    borderTopWidth: 1,
    borderTopColor: 'rgba(255,255,255,0.10)',
  }
});
```

## UX ПАТТЕРНЫ

```
✓ Swipe-to-delete в корзине (влево = удалить, красный фон)
✓ Pinch-to-zoom на фотографиях товара
✓ Skeleton loading вместо спиннеров
✓ Offline: кэш магазинов и заказов + индикатор "Нет сети"
✓ Haptic feedback при добавлении в корзину
✓ Pull-to-refresh на всех списках
✓ Shared Element Transition: карточка товара → детальная страница
✓ Bottom Sheet: корзина, детали заказа (не полный navigate)
✓ Safe area iOS/Android
✓ Локальные пуш-уведомления о статусе заказа
```
