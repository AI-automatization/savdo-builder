# Savdo — Design: Панель администратора (admin)

> Контекст для AI-генерации UI/UX панели администратора Savdo.
> Стиль: Liquid Glass / Glassmorphism. Desktop-only. Плотные данные.

---

## КОНТЕКСТ

**Приложение:** `apps/admin`
**Кто видит:** Внутренняя команда Savdo. Superadmin.
**Устройство:** Только десктоп.
**Стек:** Next.js 14 (App Router), Tailwind CSS, DaisyUI v5.
**Язык:** Русский.

**Главный принцип:** Информация важнее красоты. Плотные таблицы. Каждое действие оставляет аудит-след.
**Критично:** Деструктивные действия (бан, отклонение) → обязательное подтверждение + комментарий.
